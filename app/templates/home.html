This is a flask app to find relevant news for a particular Category and Country with filtering for keywords.

Use the search page to use a form to search for news. (This returns the results as text.)

Use the test page to get a json response from the service. (This returns the results as json.)

Use parameters as a query request on this "/test" url.

Usage: /test?country={country code}&category={category code}&filter={filter keyword}

Example: /test?country=us&category=business&filter=price
